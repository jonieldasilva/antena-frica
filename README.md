# Antena África

`Antena África` é uma apresentação web estática que traduz para o ambiente digital a proposta editorial do `Rádio África`, programa dedicado à música africana veiculado na Educadora FM 107,5, em Salvador, com retransmissão pela Rádio Nacional.

O projeto funciona como um deck interativo de apresentação, com narrativa visual e textual sobre a expansão do programa para canais digitais, especialmente `YouTube` e `Instagram`. A proposta destaca um formato de episódio semanal, desdobramentos em conteúdo social e uma linha editorial baseada em três eixos: histórico-formativo, cultural e estético-musical.

## O que existe hoje

- Uma experiência front-end em HTML, CSS e JavaScript puros.
- Navegação por slides com teclado, clique e gesto de swipe.
- Direção de arte com visual imersivo, tipografia marcante e animações sutis.
- Estrutura de apresentação para comunicar conceito, formato, linha editorial e próximos passos do projeto.

## Estrutura do projeto

- `index.html`: deck principal da apresentação.
- `style.css`: estilos, layout, animações e direção visual.
- `script.js`: navegação entre slides e interações básicas.
- `figma-plugin/`: arquivos de suporte relacionados ao plugin de Figma.

## Contexto editorial

A apresentação posiciona a `Antena África` como continuação digital do trabalho já desenvolvido no rádio, preservando a curadoria, o contexto e a conexão com a música africana. O objetivo é ampliar o alcance do conteúdo para novas audiências sem perder o rigor editorial do projeto original.

## Próximos passos mapeados no deck

## Como abrir localmente

Como o projeto é estático, basta abrir o arquivo `index.html` no navegador. Se preferir, você também pode servir a pasta com qualquer servidor HTTP simples.
